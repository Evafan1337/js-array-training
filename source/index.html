forEach

Давайте рассмотрим более интересные методы. Зачастую в практике случается так, что нам неизвестно количество элементов в массиве или наши элементы имеют более сложную структуру. В таких случаях можно использовать метод Array.prototype.forEach()

const students = [
  {
    id:1,
    name: 'Саша',
    rating:8,
    processed: false
  },
  {
    id:2,
    name: 'Миша',
    rating:5,
    processed: false
  },
  {
    id:3,
    name: 'Сережа',
    rating:7,
    processed: false
  }
]

Попытаюсь расписать задачу, похожую на практическую. У нас есть список студентов (выгрузка из условной Базы Данных). Все ученики сдали на отлично свои курсовые работы, и их рейтинг надо повысить на 1 и пометить, что их анкеты успешно обработаны. Как это сделать?

const processingForEach = (elem) => {
  console.log(`processingFunction`);
  elem.processed = !elem.processed;
  elem.rating = elem.rating + 1;
  console.log(elem);
}

students.forEach(element => processingForEach(element));
            
processingFunction
{id: 1, name: "Саша", rating: 10, processed: true}
processingFunction
{id: 2, name: "Миша", rating: 10, processed: true}
processingFunction
{id: 3, name: "Сережа", rating: 10, processed: true}

Определили анонимную функцию с именем processingForEach и передали ее в качестве аргумента для forEach. Внутри скобок текущий элемент цикла мы определили как element и передали аргументом его в processingForEach. Дальше внутри функции processingForEach мы производим необходимые манипуляции над элементом цикла. Таким образом мы перебрали все элементы массива students.

filter

Еще одна частая задача это отфильтровать массив по какому-либо критерию: по значению параметра, по дате и тд. В Javascript это реализуется с помощью Array.prototype.filter()

Применяется та же практика, что и при forEach: применяем особую функцию, заготовленную ранее и передающюся как аргумент функции filter, которая в свою очередь работает с каждым элементом массива. Только важное условие: необходимо, чтобы функция возвращался True/False. Результатом filter является массив значений (новый), в котором находятся все элементы исходного, давшие при работе функции "обработчика" True

const funcFilter = (elem) => {
  if (elem.rating > 7){
    return true;
  } else {
    return false;
  }
}
let stFilter = students.filter(item => funcFilter(item));
console.log(stFilter);
            
{id: 1, name: "Саша", rating: 9, processed: true}
{id: 3, name: "Сережа", rating: 8, processed: true}

map

С этим будет попроще. Лично по-моему мнению map очень похож на forEach, только с одной оговоркой: мы создаем новый массив, "доработанный" нашим обработчиком массивов (колбэк или анонимная функция). Ссылка на документацию: Array.prototype.map()

const processingMap = (elem) => {
  delete elem.processed;
  if (elem.rating > 7){
    elem.status = 'Отличник'
  } else {
    elem.status = 'Ударник'
  }
  return elem;
}
let studentsMap = students.map(item=> funcMap(item));
console.log(studentsMap);

{id: 2, name: "Миша", rating: 6, status: "Ударник"}

every/some

Бывают случаи когда надо проверить весь массив на некое условие. Javascript опять предлагает нам методы, основанные на колбэках. Array.prototype.some() проверяет ( возвращает True/False ), если хотя бы один раз колбэк, примененный ко всем элементам массива вернет True. Array.prototype.every() в свою очередь более строгий. Он возвращает True, если все элементы массива, обработанные колбэком вернут True. Дословно, читая названия метода можно понять как они работают. 

const funcFilter = (elem) => {
  if (elem.rating > 7){
    return true;
  } else {
    return false;
  }
}

let stEvery = students.every(item => funcFilter(item));
console.log(stEvery);
            
false
            
let stEvery = students.some(item => funcFilter(item));
console.log(stEvery);
            
true

reduce

Настал момент разобрать один из самых хитрых и сложных как по мне методов. Методом, который дарит нам необычайную гибкость при работе с массивами. В некоторых статьях указывается, что данный метод в полной мере описывает функциональное программирование. Громкие слова, но давайте приступим. Array.prototype.reduce()

Пояснение из документации: Метод reduce() применяет функцию reducer к каждому элементу массива (слева-направо), возвращая одно результирующее значение.

Параметры

  • callback - Функция, выполняющаяся для каждого элемента массива, принимает четыре аргумента:
    • accumulator - Аккумулятор, аккумулирующий значение, которое возвращает функция callback после посещения очередного элемента, либо значение initialValue, если оно предоставлено (смотрите пояснения ниже).
    • currentValue - Текущий обрабатываемый элемент массива.
    • index (необязательный) - Индекс текущего обрабатываемого элемента массива.
    • array (необязательный) - Массив, для которого была вызвана функция reduce.
  • initialValue (необязательный) - Необязательный параметр. Объект, используемый в качестве первого аргумента при первом вызове функции callback

Давайте разберем парочку примеров. Вначале будем работать с нашими студентами.

const students = [
  {
    id:1,
    name: 'Саша',
    rating:9,
    processed: false
  },
  {
    id:2,
    name: 'Миша',
    rating:6,
    processed: false
  },
  {
    id:3,
    name: 'Сережа',
    rating:8,
    processed: false
  }
]
            
let ratingSum = students.reduce((acc, item) => {
  return acc + item.rating;
},0);
console.log(ratingSum);
            
23

Получили сумму всех оценок наших студентов, вдруг в их академии есть выборка по всей учебной группе? Так называемая коллективная ответственность .Обратите внимание на то, что в данном примере упущены аргументы index, array.

let ratingAverage = students.reduce((acc, item, index, array) =>{
  acc += item.rating;
  if (index === array.length-1){
    return acc/array.length;
  } else {
    return acc;
  }
},0);
console.log(ratingAverage);
            
7.666666666666667

Таким образом мы можем подсчитать средний балл. И руководство университета сможет оценить успеваемость всей группы.

Задача, возможно используемая на кино-порталах: подсчитать количество фильмов определенного жанра. Воспользуемся методом reduce:

let genreCount = filmList.reduce((acc, item) => {
  item.genre.forEach(elem =>{
    if(!acc[elem]){
      acc[elem] = 1;
    } else {
      acc[elem] = acc[elem] + 1;
    }
  })
  return acc;
}, []);
console.log(genreCount);

Аниме: 1
Биография: 1
Боевик: 2

Мы имеем доступ к элементу (в нашем случае нам нужен массив жанров). Проводим перебор жанров по forEach. Если в acc отсутствует поле "жанра", то создаем его со значением 1 ( появляется новый жанр, иначе просто + 1 к нашему счетчику. Обратите внимание на то, что изначальное значение acc мы обьявляем как пустой массив.
